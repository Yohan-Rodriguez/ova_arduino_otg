Glosario

  • Arduino IDE: Es el entorno de desarrollo integrado oficial para programar placas Arduino.
  • Convertidor OTG: Significa "On-The-Go" y es un dispositivo pequeño que te permite conectar dispositivos USB a tu teléfono inteligente o tableta Android.
  • Memorias USB: Son dispositivos de almacenamiento de datos portátiles que utilizan memoria flash para almacenar datos.
  • Puerto USB: Es una interfaz de entrada/salida universal que permite la conexión de dispositivos electrónicos
  • Placa Arduino: Es una placa de circuito impreso que contiene los componentes electrónicos necesarios para programar y ejecutar un proyecto Arduino.
  • Programación: Es el proceso de crear instrucciones para que una computadora o dispositivo electrónico realice una tarea específica.
  • Sensor: Es un dispositivo que detecta y responde a cambios en su entorno
    Software: Es un conjunto de instrucciones y datos que le permiten a una computadora o dispositivo electrónico realizar una tarea específica.
    Teléfono inteligente: Es un teléfono móvil con un sistema operativo avanzado que le permite realizar una amplia gama de tareas, como navegar por internet, usar aplicaciones, jugar juegos y ver vídeos.
  • Transferencia de datos: Es el proceso de mover datos de un dispositivo a otro. Los datos se pueden transferir a través de una conexión física, como un cable USB, o a través de una conexión inalámbrica, como wifi o Bluetooth.
